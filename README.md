# Меридиан Ассистент

AI-бот и мульти-канальный коннектор для Contact Center Битрикс24.

## Возможности

- **Мульти-канальность** — все основные каналы коммуникации через единый Contact Center
- **AI-бот** — автоматическая обработка входящих сообщений с AI
- **CRM-интеграция** — автоматическое создание лидов, сделок, комментариев
- **Передача менеджеру** — плавная передача диалога от бота к живому оператору

## Поддерживаемые каналы

### Мессенджеры
- Telegram
- WhatsApp (Cloud API / Business API)
- Viber
- MAX (мессенджер Сбера)

### Социальные сети
- ВКонтакте (сообщения сообщества)
- Instagram (Direct Messages)
- Одноклассники (сообщения)

### Классифайды
- Авито (сообщения с объявлений)
- Юла

### Веб-каналы
- Онлайн-чат / виджет на сайте
- Email (обращения поддержки)
- SMS

### Платформы
- Яндекс.Диалоги (Яндекс.Карты / Яндекс.Бизнес)
- Google Business Messages

## Архитектура

```
Клиент → [Канал] → Сервер (коннектор) → Bitrix24 Contact Center
                                              ↓
                           ┌──────────────────┼──────────────────┐
                           ↓                  ↓                  ↓
                      AI-бот (авто)      Создание лидов     Передача
                                         в CRM              менеджеру
```

## Установка

1. Зарегистрируйте приложение в [Кабинете разработчика Bitrix24](https://vendors.bitrix24.com/)
2. Получите OAuth-ключи (client_id, client_secret)
3. Установите приложение на ваш портал Битрикс24
4. Настройте коннекторы в разделе Contact Center

## Требования

- PHP 8.0+
- Bitrix24 (Cloud или On-Premise)
- OAuth-приложение с правами: `im`, `imconnector`, `imopenlines`, `imbot`, `crm`, `contact_center`, `user`
